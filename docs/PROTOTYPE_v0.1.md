## 1. 页面导航总览

### 1.1 整体流程

登录页 -> 开发者控制台 (super_admin) / 猫咪CRM系统 (商家员工)

### 1.2 开发者控制台导航

控制台首页 -> 商家管理(列表/创建/详情含模块配置) / 模块管理 / 系统配置 / 审计日志 / 数据统计

### 1.3 CRM系统导航

CRM仪表盘 -> 猫咪管理(列表/详情) / 客户管理(列表/详情) / 销售管理(预订/合同) / 健康管理 / 财务 / 分店管理 / 员工管理 / 个人设置

## 2. 公共页面

### 2.1 登录页

URL: /cat/login

顶部: Cat_Sys Logo + "猫咪CRM管理系统"
中部: 手机号/用户名输入框 / 密码输入框 / 登录按钮 / "还没有账号？立即注册"链接
底部: 版权信息

功能: 自动识别用户类型跳转(超管->控制台, 商家员工->CRM)

### 2.2 注册页

URL: /cat/register

表单字段: 手机号* / 验证码(预留) / 猫舍名称* / 联系人* / 联系电话* / 密码* / 确认密码*
底部: "注册并开通"按钮 / "已有账号？去登录"链接

功能: 模式A注册即开通, 提交后自动登录跳转CRM仪表盘

## 3. 开发者控制台 (Super Admin)

### 3.1 控制台首页

左侧: 固定导航菜单(控制台/商家管理/模块管理/系统配置/审计日志/数据统计)
顶部: 用户信息 + 退出
内容: 四个统计卡片(总商家/活跃/即将到期/本月新增) + 近期动态列表 + 快速操作按钮

### 3.2 商家列表

搜索框(猫舍名/联系人/手机号) + 状态Tab切换(全部/活跃/已停用/已到期) + 表格(ID/名称/联系人/状态/有效期/操作) + 分页 + "新增商家"按钮

### 3.3 创建商家

基本信息区: 猫舍名* / 营业执照 / 联系人* / 电话* / 邮箱 / 地址
账号信息区: 登录手机号* / 初始密码*
有效期: 过期日期选择器
高级模块: 全部默认勾选, 可取消
保存/取消按钮

### 3.4 商家详情

摘要卡片(名称/状态/联系人/有效期/剩余天数) + 操作按钮(编辑/停用/删除/续期)
Tab切换: [基本信息] [模块配置] [分店列表] [操作日志]
模块配置Tab: 基础模块灰色锁定 + 高级模块勾选框 + "保存配置"按钮

### 3.5 其他页面

模块管理: 模块列表(编码/名称/类型/排序/开通商家数)
系统配置: 注册开关/新商家默认有效期/默认模块
审计日志: 筛选+表格(时间/操作人/商家/操作类型/详情)
数据统计: 图表(商家增长趋势/模块使用率等)

## 4. 猫咪CRM系统 (商家/分店)

### 4.1 全局布局

顶部栏: [Logo + Cat_Sys] [分店切换下拉框 v] [用户名(角色) v] [退出]
左侧导航: 仪表盘/猫咪管理/客户管理/销售管理/健康管理/财务/分店管理/员工管理/设置 (按角色和模块开通动态显示)
内容区: 页面标题 + 筛选+操作栏 + 数据内容

### 4.2 CRM仪表盘

统计卡片: 猫咪总数/客户总数/预订进行中/待办任务数
近期待办: 下次驱虫/下次疫苗/客户回访等
最新动态: 最近操作记录(谁+什么时间+做了什么)

### 4.3 猫咪列表

筛选栏: 品种/毛色/状态/性别下拉 + 搜索框(名字/芯片号)
表格: 缩略图/名字/品种/毛色/性别/年龄/状态/价格/操作
操作: "新增猫咪"按钮 + 每行编辑按钮
底部: "最后操作人: XXX 最后操作时间: YYYY-MM-DD HH:mm"

### 4.4 猫咪详情/编辑

左侧: 照片区(主图+多图, 最多9张)
右侧: 基本信息表单(名字/品种/毛色/性别/生日/芯片号/血统号/状态/售价/父猫/母猫)
底部: 备注文本区 + "最后操作人/最后操作时间" + 保存/取消按钮
下方: 健康记录列表 + "新增健康记录" + 销售记录
注: 销售/护理员均可修改售价

### 4.5 客户管理

客户列表: 搜索 + 状态/来源筛选 + 表格(姓名/手机/微信/来源/状态/操作) + "新增客户"
客户详情: 基本信息(姓名/手机/微信/邮箱/地址/来源/状态) + 偏好(品种/颜色/预算) + 备注 + 跟进记录 + 预订/合同记录

### 4.6 销售管理

预订列表: 状态/日期筛选 + 表格(ID/客户/猫咪/定金/总价/状态/操作) + "创建预订"
创建预订(弹出框): 客户选择/猫咪选择/预订日期/定金金额/备注 + 提交/取消
合同管理: 从预订生成合同, 表格查看

### 4.7 分店管理 (商家主理人)

表格: ID/名称/地址/电话/员工数/状态/操作
操作: "+新增分店" / 编辑 / 删除(二次确认)

### 4.8 员工管理 (商家主理人)

筛选: 分店/角色/状态
表格: ID/姓名/手机/分店/角色/状态/操作
操作: "+添加员工"弹出框(姓名*/手机号*/分店*/角色*/职位/初始密码*) / 离职(禁用) / 启用

### 4.9 健康管理 (Pro模块)

筛选: 猫咪/类型/日期
表格: ID/猫咪/类型/日期/下次日期/医生/操作
近期提醒: 下次疫苗/驱虫日期的倒计时提醒
操作: "+新增记录"按钮

## 5. 页面元素规范

1. 顶部栏: Logo左, 分店切换中, 用户名角色+退出右
2. 侧边导航: 220px宽, 图标+文字, 高亮当前, 按角色过滤
3. 操作反馈: 成功绿色, 失败红色, 删除二次确认, 提交Loading
4. 最后操作人/时间: 列表页表格底部, 详情页表单底部
